Opleiding

Software Developer
Davinci College Gorinchem. September 2022 - Heden

De opleiding die ik doe is Software Developer. In deze opleiding heb ik al veel geleerd en gemaakt. Ik kwam hier binnen met 0 kennis en nu ben ik al redelijk ver. Ik heb al veel geleerd en een paar opdrachten sprongen er wel uit waar ik veel van geleerd heb. Je kunt die opdrachten bekijken door op het Github icoontje te klikken in de balk beneden in het scherm.
Ik wil graag een front-end Developer worden. Dus graag het uiterlijk en de technische dingen maken. Ik was bij deze opleiding gekomen omdat ik wilde weten hoe een website een app of bijv. een game gemaakt werd vanaf code. En hoe het dan eruit kwam te zien en hoe je dat aanpast. Daarom wil ik daar iets mee gaan doen omdat ik het leuk en interessant vind.
